PR title example: Align oracle enhancement benchmark with shared beamforming path

Concise description:
Add an oracle-steered XVF3800 enhancement benchmark that reuses the shared realtime beamforming family (`delay_sum`, `mvdr_fd`, `lcmv_null`) instead of a benchmark-local beamformer path.

PR title example: Add shared output enhancer tuning for oracle beamforming runs

Concise description:
Add a shared realtime output-enhancer path for beamformed mono audio, including the existing Wiener-style postfilter plus RNNoise and Wiener+RNNoise modes, and expose the enhancer controls through the runtime request/config layer. Add a CPU-focused Optuna tuner that caches oracle-steered synthetic `lcmv_null` outputs, searches denoiser settings under an RTF target, persists the Optuna study to SQLite, and writes best-parameter summaries. Also add support for running the enhancer at an internal frame size smaller than the beamformer hop so `50 ms` beamforming can still use `10 ms` denoising, and validate that path with targeted safety tests and subset reruns.

None.
